핵심 기능

게임사에 필요한 모든 것을 하나의 플랫폼으로

Secure RAG 지식 검색 & 문서 생성

보안 걱정 없이 사내 문서를 AI가 학습합니다. 개발 환경은 Public AI API로, 운영은 내부 LLM으로 즉시 전환 가능한 구조로 설계되었습니다.

인게임 데이터 수집

크래시 로그, 유저 피드백을 REST API로 수집합니다. 정형/비정형 데이터를 모두 처리하며 순간적인 트래픽도 안정적으로 처리합니다.

인텔리전트 이슈 트래커 & 대시보드

실시간 버그 현황을 시각화하고 칸반 보드로 이슈를 관리합니다. 임계값 초과 시 Slack 알림도 자동 발송됩니다.

지식 자산화 & 패치노트 자동 생성

해결된 버그 티켓과 수정된 기획서를 종합하여 유저 친화적인 패치노트를 자동으로 생성합니다.
